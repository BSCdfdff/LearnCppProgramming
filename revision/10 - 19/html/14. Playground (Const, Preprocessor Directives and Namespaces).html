Messing with¶

  • Const, Main point here. It comes in two variants. 1) Normal const, and other Preprocessor constant or Directives. 2nd one is done before compile time, and is global. Also we see different formats fo Preprocessor Directives

  • Constant pointers. Main essence here: It comes in 3 flavours: 1, 2, 3, with last one a combination of first two.

$$ \newcommand\T{\Rule{0pt}{1em}{.3em}} \begin{array}{|r|r|r|r|r|r|r|r|r|r|} \hline \T&\color{green}{Data\;is\;constant}\T&\color{green}{Pointer\;is\;constant}\T&\color{green}{Pointer(Address)\;is\;constant} \\\hline \T&\color{red}{Cant\; change\; *ptr(value)}\T&\color{red}{Cant\; reassign\; pointer} \T&\color{red}{Cant\; change\; pointer(Address)}\\\hline \text{int const *ptr = &x;}\T&\Large\color{\green}{\unicode{10003}}\T&\Large\color{\red}{\unicode{10008}}\T&\Large\color{\red}{\unicode{10008}} \\\hline \text{int *const ptr = &x;}\T&\Large\color{\red}{\unicode{10008}}\T&\Large\color{\green}{\unicode{10003}}\T&\Large\color{\green}{\unicode{10003}} \\\hline \text{const int *const ptr = &x;}\T&\Large\color{\green}{\unicode{10003}}\T&\Large\color{\green}{\unicode{10003}}\T&\Large\color{\green}{\unicode{10003}} \\\hline \end{array} $$
  • Making Functions Constant Main point here is that all variables in function cannot be changed

  • Making Parameters of Function Constant Main point here: especially with parameter pointer by reference, where pointer variable is an alias of pointer passed as parameter to function, you sometimes dont want to change ths value inside function. This is reason why you need int const &j

  • Namespaces, Main point here is to use functions (classes) with same name in code, and to differentiate it from other with same name, we add "namespace" in front of it.

Pointer 1 -¶

  1. Data is constant
  2. Can reassign pointer
In [4]:
int const *p=&x;

cout<<p<<endl;

//++*p; NOT ALLOWED - DATA IS CONSTANT 

++p; //ALLOWED TO MODIFY POINTER ADDRESS
cout<<p<<endl;

int z = 20;
p = &z;     //ALLOWED TO REASSIGN POINTER, BUT CANNOT UPDATE VALUE ONCE IS IT ASSIGNED

cout<<*p<<endl;
