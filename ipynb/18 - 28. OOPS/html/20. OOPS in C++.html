public:

    }

Pointer in HEAP

How does it work¶

Then we DECLARE a pointer of type Rectangle, *p

Rectangle *p

No need to define a data varianle first

Then we INITIALIZE te pointer using new keyword, to signify the pointer needs to be in HEAP

p = new Rectangle

We could have also written it in ONE line

Rectangle *q = new Rectangle()

The rest is the same as STACK, logic

Onlt here we need to deallocate memory, when we done with the variables

Lets recap...¶

Declaring the variable (object) in STACK:

Rectangle r

Declaring the pointer for data variable (address) in STACK:

Rectangle r
Rectangle *p;
p = &r

Declaring the pointer in HEAP:

Rectangle *q = new Rectangle()
