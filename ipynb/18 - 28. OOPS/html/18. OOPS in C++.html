class Rectangle{
public:    
    int length;
    int breadth;
    
    int area(){
        return length * breadth;
    }
    
    int perimeter(){
        return 2 *( length + breadth);
    }
    
}

How much memory space (bytes) will this object occupy?¶

We need to look at the data members (the data variables defined in class) that is used in class:

int length;
int breadth;
int area(){..}
int perimeter(){}

int takes 4 bytes x 2 (lenght and breath) = 8 bytes.

Note functions will not occupy any memeory space.

So size of this class (Rectangle) = 8 bytes.

r1 = 8 bytes
r2 = 8 bytes

Note the class, which is the blueprint, does NOT occupy any memory, only the instance, i.e the object occupies memory

We use the dot (.) operator
