Pass by Value

How does it work?¶

x and y are called Actual Parameters (Arguments) a and b are called Formal Parameters (Arguments)

When pass by value, formal paremeters cannot change actual parameters

x,y has its own memory space

When do we use Call by value: When you need to do some work in swap (called procedure), and then just return. If you do return woth a vlaue, then then that value is returned. But as soon as "swap" finishes, all memory (a, b) gets dealllocated.

    a=b;
    b=temp;
}
