Global and Local Variables¶

  • Global variable cab be accessed in any function
  • Global variable is keeps it value, i.e. is like a running total
  • Global variable REMAIN in memory as long as program us running
  • Local variable, local to function, but REMOVED from memory when function exits

How does it work¶

The main take away from global and local variables is, that *global variables will remain using memory and is accessible from all parts of the program, as long as the program is running. Whereas, loval variables is only accessible from within the defining function, and will be removed from memorym when the function exists.

But where is Global variables? As opposed to local variables which is define in STACK, the Global variables are defined in CODE section* of memory
